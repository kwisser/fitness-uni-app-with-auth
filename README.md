# Fitness Uni App

## How to start

### Server

Switch into server folder \
`cd server` \
Install dependencys \
`npm install` \
Start fitness server \
`node app.js fitness`

### Frontend

Switch into frontend folder \
`cd frontend` \
Install dependencys \
`npm install` \
Start react app \
`npm start`
